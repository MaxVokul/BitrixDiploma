CSS Grid Row Gaps

How to set the top gap to 0 and maintain row gaps of 20px

Original Code

This had a top margin of 2dvh which created space above the grid:

.row {
  margin: 2dvh 0 0 0;
}

Solution

Remove the top margin to eliminate the upper gap:

.row {
  margin: 0;
  grid-row-gap: 20px;
}

How This Works

The original code had margin: 2dvh 0 0 0; which created space above the entire grid container.

By changing this to margin: 0;, we eliminate the top margin while keeping the row gaps at 20px through the grid-row-gap: 20px; property.

The grid maintains:

  • No gap above the first row
  • 20px gap between rows
  • 20px gap between columns (from grid-column-gap: 20px;)
